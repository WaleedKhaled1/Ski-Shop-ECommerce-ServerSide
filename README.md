# Ski- Full-Stack E-Commerce Web Application 

Welcome to **SkiNet**, a comprehensive e-commerce solution built with a decoupled architecture. 

## 🔗 Repository Links
To keep the codebase clean and modular, the project is split into two separate repositories:
* **Backend (.NET API):** You are currently in this repository.
* **Frontend (Angular Client):** [Click Here to View Frontend Repository](https://github.com/WaleedKhaled1/Ski-Shop-ECommerce-ClientSide)

## 🛠️ Tech Stack
* **Frontend:** Angular, TypeScript, Angular Material, Tailwind CSS, RxJS, Signals.
* **Backend:** .NET Web API, Entity Framework Core, SQL Server, Repository Pattern.
